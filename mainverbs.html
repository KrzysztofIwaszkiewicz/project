Hablar, ser, estar. Poznajemy podstawowe czasowniki

W j. hiszpańskim najczęściej używanymi czasownikami są hablar (mówić), ser / estar (być), tener (mieć). Poniżej przedstawiam jak się je odmienia.

Hablar
Yo - hablo / Tu - hablas / El/Ella/Usted - habla / Nosotros - hablamos / Vosotros - hablais / Ellos/Ellas/Ustedes - hablan
Ser
Yo - soy / Tu - eres / El - es / Nosotros - somos / Vosotros - sois / Ellos - son
Estar>
Yo - estoy / Tu - estas / El - esta / Nosotros - estamos / Vosotros - estais / Ellos - estan
Tender>
Yo - tengo / Tu - tienes / El - tiene / Nosotros - tenemos / Vosotros - teneis / Ellos - tienen
